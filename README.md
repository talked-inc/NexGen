NexGen is our new API for talked. More decentralized, faster and more secure.

Documentation: [APIv3](https://nexgen.talked.ga/docs)

## Installation

Use the package manager [wget](https://pip.pypa.io/en/stable/) to install foobar.

```bash
wget https://nexgen.talked.ga/download/?type=binary?no-cache=true && unzip nexgen-api.zip && sudo npm -i
```

## Before running NexGen for the first time, please do the following:
```bash
sudo npm -i
sudo npm run controlRun
npm run autoUpdate --fix-missing
```

## Usage

```bash
sudo npm run nexgenBoot
```
